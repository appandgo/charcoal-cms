Class AbstractSection

A Section is a unique, reachable page.

Types of sections

There can be different types of ection. 4 exists in the CMS module: - blocks - content - empty - external

External implementations

Sections imlpement the following Interface / Trait: - From the Charcoal\Object namespace (in charcoal-base) - Hierarchical - Routable - From the local Charcoal\Cms namespace - Metatag - Searchable
